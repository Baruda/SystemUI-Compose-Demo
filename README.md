# SystemUI Compose Demo



## Features
- Reactive pull-down **notification shade** with spring physics animation
- **Quick Settings** panel with debounced tile clicks
- **Keyguard** (lock screen) with widgets and biometric unlock simulator
- **Heads-up notifications** with auto-dismiss
- **Navigation bar** (gesture pill + 3-button mode)
- **Full Recents screen** with task cards
- **Split-screen multitasking**
- **Volume** and **Power menus**
- **Material You** dynamic theming
- **Edge-to-edge** display
- Accessibility support (semantics, content descriptions)
- Unit and Compose UI tests
